# FILA React Project

기존 FILA 웹사이트 디자인과 기능을 유지하면서 Vite + React 기반으로 재구성한 프로젝트입니다.

## 실행 방법

```bash
npm install
npm run dev
```

## 주요 페이지

- `/` 메인 페이지
- `/products` 상품 목록
- `/products/1` 상품 상세
- `/cart` 장바구니
- `/qna` Q&A CRUD 게시판

## 평가 항목 적용

- Vite + React 프로젝트
- React Router를 이용한 페이지 이동
- Header, Footer, LegacyPage, Q&A 컴포넌트 구조
- `useState`를 이용한 Q&A 데이터 및 화면 상태 관리
- `props`를 이용한 컴포넌트 데이터 전달
- 클릭, 입력, 등록, 수정, 삭제 이벤트 처리
- Q&A Create / Read / Update / Delete
- 문의 상세 조회 시 조회수 증가
- localStorage를 이용한 Q&A 데이터 유지
- 이미지, 스타일, 컴포넌트, 페이지 폴더 구조화

## 제출

프로젝트 폴더에서 `node_modules` 폴더만 제외하고 압축하여 제출합니다.
